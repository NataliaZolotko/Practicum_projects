### Атоматизация тестов для приложения по заказу бургеров

### Цель
Протестировать программу, которая помогает заказать бургер в Stellar Burgers

Протестировать эндпоинты API для Stellar Burgers.

Написать автотесты для UI

### Что было сделано
Реализовала для веб-приложения для заказа бургеров покрытие кода юнит-тестами с применением моков и параметризации

Провела интеграционное тестирование API-эндпоинтов через Postman, включая проверку статус-кодов,структуры JSON-ответов и корректности обработки данных. 

Разработала автоматизированные UI-тесты с использованием паттерна Page Object Model для повышения читаемости, устойчивости тестов и упрощения их дальнейшей поддержки.

Обеспечила кроссплатформенную стабильность продукта, проведя тестирование верстки и функциональности в различных браузерах. 

Сгенерировала отчеты о результатах прогонов с помощью Allure.

### Используемые инструменты
Стек: Python, PyTest, Requests, Selenium, Page Object Model, Allure, Git, Chrome DevTools
