### Тестирование API приложения для заказа самокатов

### Цель
Протестировать ручки и проверить, что они корректно работают и выдают нужные ошибки.

### Что было сделано
Протестированы ручки API:
- создание курьера
- логин курьера
- создание заказа
- список заказов

По результатам тестов сгенирирован Allure отчет

### Используемые инструменты
Стек технологий:

Язык программирования: Python 3.9+

Тестирование UI: Pytest (фреймворк), Selenium (автоматизация браузера)

Тестирование API: Requests (HTTP-клиент для проверки бэкенда)

Тестовые данные: Faker (генерация случайных имён, адресов, телефонов)

Отчётность: Allure (с интеграцией allure-pytest)
