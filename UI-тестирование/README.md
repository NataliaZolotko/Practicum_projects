### Проектирование автотестов для приложения Яндекс Самоката

### Цель
Протестировать выпадающий список в разделе «Вопросы о важном» в веб‑приложении.
Протестировать "Заказ самоката".

### Что было сделано
Создан отдельный класс с Page Object
Написаны тесты на Selenium, используя параметризацию.
Тесты разделены по тематике.


### Используемые инструменты
### pytest==8.4.2:
Основной фреймворк для написания и запуска тестов. Управляет выполнением тестовых сценариев.
### selenium==4.38.0	      
Инструмент для автоматизации действий в браузере. Позволяет кликать по кнопкам, заполнять формы, проверять текст на странице.
### allure-pytest==2.15.0	 
Интеграция Allure с Pytest. Позволяет собирать данные о прохождении тестов для формирования красивых отчетов.
